Lesson 2: Getting Input from the User

Now that you know how to use variables, it’s time to make your programs interactive! 🧑‍💻

🔍 What is cin?

cin (short for "character input") allows you to receive data from the keyboard. It's like your program asking the user: "Hey, can you tell me something?" 👂

🛠️ Syntax

This is how you use cin:

    string name;
        cout << "What's your name? ";
        cin >> name;

Let’s explain:

  • cout: Asks the question
  • cin: Waits for the user to type something
  • >>: Passes the input into the variable

✅ Full Example

This program asks for your name, age, and favorite color, then responds to you:

    #include <iostream>
        #include <string>
        using namespace std;
    
        int main() {
            string name, color;
            int age;
    
            cout << "What's your name? ";
            cin >> name;
    
            cout << "How old are you? ";
            cin >> age;
    
            cout << "What's your favorite color? ";
            cin >> color;
    
            cout << "Hello, " << name << "! You are " << age 
                 << " years old and love the color " << color << "." << endl;
    
            return 0;
        }

⚠️ Tip

cin only reads one word at a time. So if your name is "Alex Popescu", it will only take "Alex". To read full lines (including spaces), you’ll learn about getline() later.
